# Security Policy

## Supported Versions

Security fixes target the latest released version.

## Reporting a Vulnerability

Please open a private security advisory on GitHub if the repository has advisories enabled. If not, contact the maintainer using the security contact listed on the repository profile.

Do not publish working exploits for token theft, local account scraping, or Discord IPC abuse before a fix is available.

## Data Handling

Codex Discord RPC does not read Codex auth files, Discord token files, browser profiles, or ChatGPT account storage. Subscription type and work status are explicit user-provided values.
