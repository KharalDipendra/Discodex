# Contributing

Thanks for helping improve Codex Discord RPC.

## Local Setup

```powershell
npm install
npm run build
npm test
npm run check
```

## Pull Requests

- Keep changes focused.
- Add or update tests for behavior changes.
- Keep source changes in TypeScript under `src/`.
- Update the README when user-facing config or CLI behavior changes.
- Avoid adding account scraping, token reading, or hidden telemetry.

## Release Notes

Use clear, practical language. Call out breaking changes, new config options, and migration steps.
